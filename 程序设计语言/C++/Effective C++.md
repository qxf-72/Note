# 01：将 C++视为语言联邦

将 C++视为一个由相关语言构成的语言联邦而非单一语言：
- C：基本语法
- Object-Oriented C++：面向对象的部分
- Template C++：关于模板的泛型编程部分
- STL

<span style="background:#fff88f">C++高效编程守则视情况而定，取决于使用 C++的哪一部分</span>。
